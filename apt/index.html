luksmith apt repository

Org-grade LUKS management for Ubuntu 22.04/24.04. Quick setup:

curl -fsSL https://cdmx-in.github.io/luksmith/setup.sh | sudo sh
sudo apt install luksmith

Or manually:

sudo curl -fsSL https://cdmx-in.github.io/luksmith/luksmith-archive-keyring.gpg \
  -o /usr/share/keyrings/luksmith-archive-keyring.gpg
echo 'deb [signed-by=/usr/share/keyrings/luksmith-archive-keyring.gpg] https://cdmx-in.github.io/luksmith ./' \
  | sudo tee /etc/apt/sources.list.d/luksmith.list
sudo apt update && sudo apt install luksmith

Updates arrive via normal apt upgrade. Source: github.com/cdmx-in/luksmith
